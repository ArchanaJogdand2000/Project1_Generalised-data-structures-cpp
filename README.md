# Project1_Generalised-data-structures-cpp
Generic C++ library implementing linear and non linear data structures using object-oriented design principles.


This repository contains C++ implementations of generic data structures.
It includes arrays, linked lists, stacks, queues with reusable templates. 
## Features :
- Linear and non-linear data structures 
- Generic template-based implementation
- Object-oriented design principles
  
## How to Run
1. Clone the repository
2. Compile using g++
3. Run the executable
